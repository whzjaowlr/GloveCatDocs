# 보상 시스템

다양한 활동을 통해 GCAT 보상을 받으세요.

## 일일 체크인

### 규칙

- 하루에 한 번 체크인 가능
- UTC 00:00 기준 리셋
- 연속 체크인으로 스트릭 보너스

### 스트릭 보너스

| 연속 일수 | 보너스    |
| --------- | --------- |
| 1-6일     | 기본 보상 |
| 7일       | **+50%**  |
| 14일+     | 추가 보상 |

### 사용 방법

1. 보상 탭 이동
2. "체크인" 버튼 클릭
3. 트랜잭션 승인
4. 보상 수령!

## 리더보드

### 시즌 보상

| 순위   | 보상       |
| ------ | ---------- |
| 🥇 1위 | 5,000 GCAT |
| 🥈 2위 | 3,000 GCAT |
| 🥉 3위 | 2,000 GCAT |

### 점수 획득

- 일일 체크인
- 스테이킹

## 업적 시스템

특별한 활동을 달성하면 업적 배지와 보상을 받습니다.

| 업적        | 조건        | 보상        |
| ----------- | ----------- | ----------- |
| First Stake | 첫 스테이킹 | 배지 + GCAT |
| Voter       | 첫 투표     | 배지        |
| Collector   | NFT 보유    | 배지        |

## FAQ

### Q: 체크인을 놓치면 스트릭이 초기화되나요?

A: 네, 하루라도 놓치면 스트릭이 0으로 초기화됩니다.

### Q: 리더보드 시즌은 얼마나 되나요?

A: 시즌은 약 30일입니다.
