# NFT 가이드

GloveCat NFT를 보유하면 다양한 혜택을 받을 수 있습니다.

## 티어 시스템

| 티어        | 수수료 할인 | 스테이킹 부스트 |
| ----------- | ----------- | --------------- |
| 🥉 Bronze   | 5%          | 1.1x            |
| 🥈 Silver   | 10%         | 1.2x            |
| 🥇 Gold     | 15%         | 1.3x            |
| 💎 Platinum | 20%         | 1.5x            |
| 👑 Diamond  | 30%         | 2.0x            |

## 혜택

### 수수료 할인

NFT를 보유하면 거래 수수료가 할인됩니다.

### 스테이킹 부스트

스테이킹 보상에 배율이 적용됩니다.

### 투표권 부스트

거버넌스 투표 시 추가 투표권을 받습니다.

### 독점 혜택

- 신규 기능 얼리 액세스
- 에어드롭 우선권
- 커뮤니티 이벤트 참여

## 획득 방법

1. **민팅**: 공식 민팅 이벤트
2. **마켓플레이스**: OpenSea 등에서 구매
3. **에어드롭**: 커뮤니티 이벤트
