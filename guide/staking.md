# 스테이킹 가이드

GCAT을 스테이킹하여 보상을 받는 방법입니다.

## 스테이킹 종류

### Flexible 스테이킹

| 특징      | 설명            |
| --------- | --------------- |
| 입출금    | 언제든 자유롭게 |
| APY       | 기본 APY 적용   |
| 최소 금액 | 없음            |

::: tip 장점 유동성을 유지하면서 보상을 받을 수 있습니다. :::

### Lock-up 스테이킹

장기 스테이킹으로 더 높은 APY를 받으세요.

| 기간  | APY | 비고     |
| ----- | --- | -------- |
| 30일  | 1%  | -        |
| 90일  | 3%  | -        |
| 180일 | 5%  | -        |
| 365일 | 8%  | -        |
| 2년   | 12% | -        |
| 4년   | 15% | 최대 APY |

::: warning 주의 Lock-up 기간 동안에는 출금이 불가능합니다. :::

## 사용 방법

### 1. 지갑 연결

1. dApp 접속: [app.glovecat.com](https://app.glovecat.com)
2. "지갑 연결" 버튼 클릭
3. MetaMask 또는 Rainbow 지갑 선택
4. Base 네트워크 확인

### 2. 스테이킹

1. 스테이킹 탭 이동
2. Flexible 또는 Lock-up 선택
3. 스테이킹 금액 입력
4. Lock-up의 경우 기간 선택
5. "스테이킹" 버튼 클릭
6. 지갑에서 트랜잭션 승인

### 3. 보상 청구

1. "보상 청구" 버튼 클릭
2. 트랜잭션 승인
3. 보상 수령 완료

## FAQ

### Q: 언스테이킹 시 수수료가 있나요?

A: 없습니다. 가스비만 지불하시면 됩니다.

### Q: 보상은 얼마나 자주 받을 수 있나요?

A: 언제든 원할 때 청구할 수 있습니다.
